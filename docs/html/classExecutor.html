Detailed Description

template<typename ReturnType, typename ArgType>
class Executor< ReturnType, ArgType >

Concurrent execution of tasks using fixed threads pool.

Similar to Executor<void,void> but here functions to execute must have a parameter and return something.

Template Parameters
ReturnTypeReturn type of tasks to perform. Not void.
ArgTypeType of the parameter the tasks take. Not void.
