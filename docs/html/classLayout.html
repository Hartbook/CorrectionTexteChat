Detailed Description

Transfers the punctuation from one file to another.

Example :

if file1 contains "[06:20:44] _CLIENT_ : Sesi est une frase... pleines de fotes !" and file2 contains "ceci est une phrase pleine de fautes"

Layout layout(file1, file2);
File file3("path/to/file3.txt", "w");
layout.transferLayout(file3);

Will produce a file containing "[06:20:44] _CLIENT_ : ceci est une phrase... pleine de fautes !"

The purpose is to re-introduce the syntax from original un-corrected text to the syntax-less corrected text.
