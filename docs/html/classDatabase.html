Detailed Description

Contains necessary data to perform a correction.

Creation of the Database is a long process, that is why you only need to build it once. During the build process several files representing the Database will be written. Database can then be directly loaded from those files.

When constructed the Database represents nothing and must not be used as it is. You have to call buildFromCorpus or readFromFiles in order to initialize it.

Build the Database, long process.

Construct Lexicon, GramsCounter, TranslationTable using the files passed as parameters. Creates a number of files, that can later be used for readFromFiles().

Parameters

corporaList of paths to files that contain text to learn ngrams from.
pairsList of pairs of paths to files consisting of un-corrected and corrected versions of the same text.
