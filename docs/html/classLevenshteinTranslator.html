private

Return the cost of substituing i-th element of s1 by the j-th element of s2.

If it returns 1 if s1[i] != s2[j] and 0 otherwise, getProximity() will return the edit distance. However as return value is float it can be tuned to penalize unlikely substitutions.

Parameters
s1String containing the original letter.
s2String containing the transformed letter.
iIndex of the original letter.
jIndex of the transformed letter.
Returns
0 if s1[i] != s2[j], the more it's high the less likely the substitution is.
